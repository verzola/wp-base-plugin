# Base Plugin

## Features
- Composer
- PSR-4 Autoloading
- PHPUnit
- PHPCSFixer
- PHPDI

## Requirements
- PHP
- Composer

## Setup
```sh
$ git clone git@bitbucket.org:webfatorial/wp-plugin-base.git
$ mv wp-plugin-base <wp folder>/wp-content/plugins
$ cd <wp folder>/wp-content/plugins/wp-plugin-base
$ composer install
```
